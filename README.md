# ドローン見積・書類作成アプリ

ドローン業務の個人事業主・小規模事業者向けに、案件ごとの「見積書・作業計画書・安全チェックリスト・顧客提出メール文」を自動生成する Web アプリです。書類作成時間の削減を目的としています。

## 概要

- **対象**: ドローン業務（屋根点検・外壁点検・橋梁点検・太陽光点検・空撮・測量など）
- **提供機能**: 顧客・案件・飛行条件・料金を入力し、4 種類の文書をタブで表示・コピー。見積書は印刷しやすいレイアウトに対応。
- **データ保存**: ブラウザの localStorage に案件を保存。編集・複製・削除が可能。
- **初回利用時**: サンプル案件が 2 件自動で入り、そのまま編集・出力の動作確認ができます。

## 機能一覧（MVP）

1. **案件一覧（トップ）**
   - 保存済み案件のカード表示（現場名・顧客名・業務種別・作業予定日・税込合計）
   - 新規案件・編集・複製・削除

2. **案件入力**
   - 顧客情報（顧客名・担当者・メール・電話）
   - 案件情報（現場名・住所・業務種別・作業予定日・機体・人数・日数・備考）
   - 飛行・現場条件（DID・目視外・夜間・道路上空・第三者立入管理・周辺状況・離着陸場所・補助者・安全特記事項）
   - 料金（基本・出張・オプション・値引き・消費税・合計自動計算）

3. **出力**
   - **見積書**: 印刷用レイアウト、コピー・印刷ボタン
   - **作業計画書**: 案件概要・実施日時・場所・作業内容・機材・人員・リスク・安全対策・注意事項
   - **安全チェックリスト**: 気象・機体・バッテリー・第三者・経路・緊急時・補助者・法規制の 8 項目
   - **顧客提出メール文**: 宛名・挨拶・見積送付案内・実施内容要約・確認依頼・署名

## 動作確認の手順（ダミーデータ）

1. `npm run dev` で起動し、トップを開く。
2. 初回はサンプル案件が 2 件表示される（ストアが空のときのみ自動投入）。
3. いずれかのカードで「編集」→ 内容を確認・変更し「保存」。
4. 「出力」→ 見積書・作業計画書・安全チェックリスト・メール文のタブを切り替え、各「コピー」でクリップボードにコピーできることを確認。
5. 見積書タブで「印刷」を押し、見積書部分のみが印刷プレビューに表示されることを確認。
6. 「新規案件」で空のフォームから入力し保存。「複製」で既存案件のコピーが作成されることを確認。

## プロジェクト構成（抜粋）

```
src/
├── app/                    # ページ（/, /case/new, /case/[id], /case/[id]/output）
├── components/
│   ├── layout/             # Header
│   ├── case/               # 案件入力フォーム（顧客・案件・飛行条件・料金）
│   ├── case-list/          # 案件一覧・カード
│   ├── output/             # 見積書・作業計画書・チェックリスト・メール View
│   └── ui/                 # shadcn/ui
├── lib/
│   ├── types/              # Customer, Pricing, Case, BusinessType
│   ├── store/              # localStorage 案件 CRUD
│   ├── calc/               # 料金計算（税抜・税込・合計）
│   ├── templates/          # 4 種類の文面生成
│   └── data/               # 初期案件・ダミーデータ
└── docs/
    └── DESIGN_PROPOSAL.md  # 画面・データ・ディレクトリ・実装ステップの設計書
```

## 今後の拡張（設計上の方針）

- **PDF 出力**: ブラウザの印刷から PDF 保存、または専用ライブラリで同じレイアウトを PDF 化。
- **顧客管理**: 顧客を独立エンティティにし、案件から参照。`/customers` と API を追加。
- **案件管理**: 一覧のフィルタ・検索・ステータス（見積中・受注・完了等）。
- **テンプレート管理**: 文面を DB または JSON で保持し、画面から編集可能に。
- **提案書・報告書・申請準備**: `/case/[id]/proposal` 等のルートとテンプレート・View を追加。

設計の詳細は `docs/DESIGN_PROPOSAL.md` を参照してください。
